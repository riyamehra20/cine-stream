#  Cine-Stream — Media Explorer

A movie discovery app built with **Next.js 15** (App Router), migrated from a Vite + React SPA to enable Server-Side Rendering (SSR) and improved SEO.

##  Live Demo
[https://your-cine-stream.vercel.app](https://cine-stream-iota-ebon.vercel.app/)

### Theme
Next.js 15, Server-Side Rendering (SSR), and SEO Optimization.

### Project
Next.js Migration — Cine-Stream SPA Upgrade.

##  Features

-  Browse popular movies from TMDB API
-  Search movies in real-time with debounce
-  Infinite scroll for loading more movies
-  Fully responsive design
-  SSR for better SEO and performance


##  Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 18
- **API:** TMDB (The Movie Database)
- **HTTP Client:** Axios
- **Deployment:** Vercel
